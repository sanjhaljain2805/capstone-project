# Deadline Manager

## Problem

Students often miss assignment and event deadlines.

## Solution

A web app to track assignments and events with deadlines.

## Features

* Add, edit, delete tasks
* Deadline tracking
* Search and filter
* Status (Overdue / Due Soon / Normal)

## Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express
* Database: MongoDB

## Flow

User → Frontend → Backend API → Database → Response → UI
